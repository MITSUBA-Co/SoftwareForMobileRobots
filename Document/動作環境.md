# ミツバ - 移動ロボット用ソフトウェア

## 3. 動作環境
### 3.1 制御コンピュータ  

* プロセッサ: Intel Core i5-1135G または同等以上の性能
* メモリ: 少なくとも 16 GB の DDR4 RAM
* ストレージ: SSD 256 GB 以上
* OS: Ubuntu 22.04  
  　　(Python3.10およびROS2 Humbleのインストールが必要)
* インターフェース：  
　　　　　 |- HDMI 出力  
　　　　　 |- USB 3.0 ポート  
　　　　　 |- Wi-Fi モジュール
